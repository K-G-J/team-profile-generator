# Team Profile Generator

## Description
A command line application to create a company profile of employees

[Demo video](https://youtu.be/1iKx_mV1UHE)

## Instructions 
    1. clone the repository
    2. open index.js 
    3. run "npm init"
    4. run "npm install inquirer"
    5. run "npm install jest"
    6. run "node index"
